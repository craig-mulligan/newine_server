newine_server
=============

Local Server for Newine dispensers
#Requisitos
Ruby 2.1.0

##Institucional
Backend encargado de administrar la información de vinos en los dispensers

##Descripción
Existen las clases `Bottle_holder`, `Category`, `Dispenser`, `Event`, `Special_event`, `Tag`, `User`, `Wine`.
